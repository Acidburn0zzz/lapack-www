* [Igor Kozachenko, Jim Demmel, UC Berkeley] *Symmetric-indefinite Factorization: New storage format for L factor in Rook Pivoting and Bunch Kaufman of LDLT, storage format akin to LU, enables Level 3 BLAS TRS and TRI*

Added routines for new factorization code for symmetric indefinite (or Hermitian indefinite) matrices
with bounded Bunch-Kaufman (rook) pivoting algorithm.
( new storage format for factors U( or L ) and D, and pivoting information IPIV ).

factor L is stored explicitly in lower triangle of A;
diagonal of D is stored on the diagonal of A;
subdiagonal elements of D are stored in array E;
IPIV format is the same as in *_ROOK routines.
